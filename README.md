# TechLaunch Nigeria

A GitHub Pages site showcasing TechLaunch Nigeria's tech training programmes — AI/ML, Automation, and App & Web3 Development.

## Pages

- **Sales Page** — full marketing landing page with hero, tracks overview, bundle deals, how it works, mentoring tiers, payment plans, FAQ, and enrol CTA
- **Training Packages** — tabbed breakdown of all three tracks (AI/ML, Automation, App & Web3) with tiered pricing, delivery details, and sample projects
- **Bundle Deals** — all-three-track bundle options (Jumpstart, Professional, Mastery) with savings, features, and revenue metrics

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Site goes live at `https://<username>.github.io/<repo-name>`
